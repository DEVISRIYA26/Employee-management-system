# Employee DBMS

A simple, interactive Employee Database Management System (DBMS) web application built with HTML, CSS (Bootstrap 5), and Vanilla JavaScript. 

## 🚀 Features
- **Employee Directory:** View a list of all employees along with their ID, Name, Email Address, and Gender.
- **Dynamic Filters:** Easily categorize and filter staff by:
  - All Staff
  - Male
  - Female
- **Real-time Search:** Instantly find employees by typing their first name in the search bar.
- **Modern & Responsive UI:** Clean, aesthetically pleasing interface built with Bootstrap 5, featuring hover animations, gradients, and a responsive layout that works on any device.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6)
- **Framework:** Bootstrap 5 (via CDN for rapid styling)
- **Icons:** Bootstrap Icons

**Expected Output:** When you open the application, you will see a modern dashboard with a welcome hero section at the top. Below that are filter buttons ("All Staff", "Male", "Female") and a search bar. The bottom section displays an interactive table listing all the employees loaded from `script-EBMS.js`. By typing in the search bar or clicking the filters, the table will instantly update its contents.
